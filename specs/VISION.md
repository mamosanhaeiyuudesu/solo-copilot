# solo-copilot — プロジェクトビジョン

**このファイルは不変です。変更する場合は重大な意思決定が必要です。**

## What（何を作るか）

個人が日常的に使えるAIエージェントアプリケーション。
Claude・OpenAI・Gemini と自然言語で対話し、タスクを自動化・補助するWebアプリ。

## Why（なぜ作るか）

- 既存のAIチャットツールへの依存を減らし、自分のデータを自分で管理したい
- Cloudflare の無料枠で低コストに運用できる個人ツールが欲しい
- 複数のAIプロバイダーを切り替えながら最適なモデルを選べる環境を作りたい
- AIエージェントパターンを実際のプロダクションで学習・実験したい

## Who（誰が使うか）

主に自分自身（個人利用）。将来的に信頼できる少数のユーザーに開放する可能性あり。

## 制約

- Cloudflare Workers の制限内で動作すること（CPU時間・メモリ）
- 月間コストは Cloudflare + AI API の合計で実用的な範囲に抑える
- データは Cloudflare D1 に保存し、外部サービスへの依存を最小化する
- ユーザーデータはサーバーサイドで暗号化して保存する
