**ステータス**: Done

## 概要

各ツールへのナビゲーションリンクを並べたトップページ。チャット・データ取込・記憶ビューアの3カードのみを表示する。

## 背景・動機

アプリ全体のエントリーポイント。サマリー情報などの表示は不要（余計な複雑さを避ける）。
2026-06 のスコープ絞り込みで「記憶ツール」に軸足を移したため、タスク管理・健康管理・財務管理のカードは削除した。

## 受入れ基準

- [x] チャット（/chat）へのリンクカードが表示される
- [x] データ取込（/import）へのリンクカードが表示される
- [x] 記憶ビューア（/memory）へのリンクカードが表示される
- [x] 記憶ビューア（/memory）へのリンクがヘッダー右にも表示される
- [x] 認証済みユーザーのみ表示される
- [x] タスク管理・健康管理・財務管理のカード（Coming Soon 含む）は表示しない

- `app/pages/index.vue`
