## 概要

各ツールへのナビゲーションリンクを並べたトップページ。実装済みのツールはリンクカード、未実装は Coming Soon 表示。

## 背景・動機

アプリ全体のエントリーポイント。フェーズが進むたびに実装済みツールのリンクを追加していく。サマリー情報などの表示は不要（余計な複雑さを避ける）。

## 受入れ基準

- [x] タスク管理（/tasks）へのリンクカードが表示される
- [x] チャット（/chat）へのリンクカードが表示される
- [x] データ取込（/import）へのリンクカードが表示される
- [x] 記憶ビューア（/memory）へのリンクがヘッダー右に表示される
- [x] 健康管理・財務管理は Coming Soon として表示される
- [x] 認証済みユーザーのみ表示される
